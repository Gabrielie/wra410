You will use HTML and CSS to improve the WRA 410 course website.

First, create a new wra410 repository on your own Github site.

Next, clone the WRA 410 website from Dr. Monberg's Github site

You will need to create a README.md file in your github repository. I will use the information from the README.md file to understand the changes you've made.

Learning Goals

  • Understand Git workflow
  • Deepen HTML command knowledge
  • Understand CSS styling
  • Understand CSS Grid Layout

Requirements

  • Make changes to the background color of a page and a div.
  • Use a new font.
  • Include at least two images.
  • Use a table to format the Grading Scale on the Assignments page
  • Reformat the Student Groups page. Use a different color for links, add a title for each group, seperate each student. Use multiple columns for the divs.
  • Add navgation to the Schedule webpage
  • Revise the rows and columns on the Schedule webpage so the information is clearer or prettier.

Grading Criteria

100-90 points
You accomplished all of the requried changes and you include some new html/css commands and/or your design does an especially good job of meeting the needs of your audience (students in this course).
89-80 points
You accomplished all of the required changes.
79-70 points
You don't accomplish all of the required changes.
